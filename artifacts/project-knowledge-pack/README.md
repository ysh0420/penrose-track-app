# Penrose Project Knowledge Pack

Upload these files into the ChatGPT Project named "Penrose Intelligence Platform".

## Files

- `penrose-system-registry.md`: current system map and boundaries.
- `penrose-ai-operating-map.md`: roles and agent workflow.
- `penrose-source-of-truth-map.md`: where current data, code, views, and handoff state live.
- `penrose-current-state-snapshot.md`: current Portfolio V2 and build-priority snapshot.
- `penrose-claude-writing-brief-template.md`: writing brief template for Claude/article work.
- `penrose-codex-autonomy-policy.md`: Codex autonomy and approval policy.
- `penrose-data-coverage-quality-layer-v1-task.md`: next Codex task brief for the data coverage layer.

## How To Use

1. Upload the full folder contents to the ChatGPT Project.
2. Tell future agents to read `penrose-system-registry.md` first.
3. Treat Supabase as data/output SSOT and GitHub as code SSOT.
4. Keep Model Portfolio V2 diagnostic only.
5. Do not run AI, trades, rebalance, portfolio changes, or publication from this pack.

This pack is an operating map. It is not the data source of truth.
